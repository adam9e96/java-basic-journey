# chapter04 - Conditional Statements
조건문은 자바 프로그램의 실행 흐름을 조절할 수 있다.

조건에 따라 다른 코드를 실행하거나, 특정 코드를 반복해서 실행할 수 있다.

## `if`문
주어진 조건(`condition`)이 `true` 일 경우에만 코드 블록을 실행한다.

```
if (condition){
    // 조건이 true일 때 실행될 코드
    }
```

## `if-else` 문
조건이 `true` 일 때와 `false` 일 때 각각 다른 코드 블록을 실행한다.

```
if (condition) {
    // 조건이 true일 때 실행될 코드
} else {
    // 조건이 false일 때 실행될 코드
}
```
